# ThreeDLikeScrollGame
3D like scroll game written by piston_window for Rust

## Control
Use only Keyboard

### In Menu and Setting Screen
* W: up
* S: down
* SPACE: select

### In Game
Player
* W: stop
* A: move to left
* D: move to right

Others
* Q: rotate box to left
* E: rotate box to right

## Running
```sh
$ cargo run
```
